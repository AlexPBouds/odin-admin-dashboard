# Odin Admin Dashboard

## Purpose

The main purpose of this project is to familiarize myself with using grid layouts to make a dashboard. The dashboard itself is not going to be responsive, as the UI design is the main thing I'm focused on. I will be following a project design file as my guide for what I'm trying to achieve visually.
